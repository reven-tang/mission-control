# Mission Control — ROADMAP

## v1.0.0 (已发布)

- 后端 FastAPI（模块化：routers/ + services/）
- 前端 Streamlit（组件化：6 个页面 + 主题系统）
- 任务看板、Agent 监控、Cron 管理、Healthcheck、Skills 评分、系统资源
- Docker + docker-compose + CLI 启动器
